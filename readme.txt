gis.py - главный файл с программой
test_time.py - тестирование на скорость работы программы
translate.py - перевод с трансклина на русский
result.csv - результат выполнения программы
testdata-small.csv - база с начальными данными
Для запуска программы в консольке используйте код gis.py -i testdata-small.csv -o result.csv.
Код хорошо работает при небольшом обьеме данных, при большых размерах нужно изменить структуру кода, что-бы чтение
данных происходило не целиком, а, например построчно.
при запуске в linux нужно удалить кодировку encoding='utf-8'
